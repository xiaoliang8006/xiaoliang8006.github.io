## 题目描述

写一个函数，求两个整数之和，要求在函数体内不得使用+、-、*、/四则运算符号。



可以用++，--自增自减符号

代码如下：

```c++
class Solution {
public:
    int Add(int num1, int num2)
    {
        if(num1<0){
            num1=0-num1;
             for(int i=1;i<=num1;++i){
                --num2;
            }
            return num2;
        }else if(num2<0){
            num2=0-num2;
            for(int i=1;i<=num2;++i){
                --num1;
            }
            return num1;
        }else{
            for(int i=1;i<=num2;++i){
                ++num1;
            }
            return num1;
        }       
    }
};
```
