## 题目描述

大家都知道斐波那契数列，现在要求输入一个整数n，请你输出斐波那契数列的第n项。

n<=39

代码如下：

```c++
class Solution {
public:
    int Fibonacci(int n) {
		if(n<=0)return 0;
		if(n==1||n==2)return 1;
		return Fibonacci(n-1)+Fibonacci(n-2);
    }
};
```
